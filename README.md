# JEAF X-Fun API README #

This repository contains the source code of JEAF X-Fun API. JEAF X-Fun provides a set of cross-functions (X-Fun) that are usually required to build enterprise applications. 

JEAF X-Fun covers things like internationalization, tracing, exception handling and many more things. 

## Links ##
For further information please refer to:

* [JEAF X-Fun Getting started](https://anaptecs.atlassian.net/l/c/8vuqYGsN)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.x-fun/jeaf-x-fun-api)
* [Javadoc](https://javadoc.io/doc/com.anaptecs.jeaf.x-fun/jeaf-x-fun-api)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
